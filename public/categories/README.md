# Category cover images (Explore Our Categories)

Put cover images here for the home page category cards.

**Folder:** `public/categories/`

**Examples:**
- `Cement.jpg`
- `CPVC-PVC.jpg`
- `Sanitary.jpg`
- `Granite-Marble.jpg`
- `Bathroom-Fittings.jpg`
- `Wall-Floor-Tile.jpg`
- `Wall-Elevation-Tile.jpg`

Then in `src/components/home/FeaturedCategories.tsx`, set each category’s `image` to the path, e.g. `image: "/categories/Cement.jpg"`.
